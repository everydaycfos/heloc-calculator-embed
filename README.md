# Everyday CFO's HELOC Calculator Embed

Static Squarespace embed for the hidden Everyday CFO's First Lien HELOC calculator page.

## Production page

Target hidden Squarespace URL:

`https://www.everydaycfos.com/heloc-calculator`

## Public embed URL

Use GitHub Pages:

`https://everydaycfos.github.io/heloc-calculator-embed/`

## Squarespace code block

Paste the contents of `squarespace-embed.html` into a Code Block on the hidden page.

This calculator is intentionally embedded in an iframe so its app styles and print/PDF styles stay isolated from Squarespace page styles.
